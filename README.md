# 🔐 AuthVault - Full Stack Authentication System

A full-stack MERN authentication app with JWT-based login system and protected routes.

## 🚀 Features
- User Signup & Login
- JWT Authentication
- Protected Dashboard
- MongoDB Atlas Integration
- Secure API Routes
- Logout functionality

## 🛠 Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: MongoDB Atlas
- Auth: JWT

## 📦 Installation

### Backend
cd server  
npm install  
node server.js  

### Frontend
cd client  
npm install  
npm run dev  

## 🌐 Deployment
- Backend: Render
- Frontend: Vercel

## Setup Instructions

### 1. Clone / Extract Project

### 2. Backend Setup
cd server
npm install

Open server.js and replace:
mongoose.connect("YOUR_MONGO_URI")

with your MongoDB Atlas URI.

Run backend:
node server.js

---

### 3. Frontend Setup
cd client
npm install
npm run dev
